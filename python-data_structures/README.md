# 0x03. Python - Data Structures: Lists & Tuples

This directory contains Python scripts that practice basic operations on lists and tuples,
such as accessing elements, updating values, iterating, and returning multiple values.

Each file focuses on a small, clear task.

---

## Usage

Make sure the files are executable:

```bash
chmod +x *.py
```
Then run any script with:
```
./file_name.py

```
Files Description

0-print_list_integer.py
Prints all integers in a list, each on a new line.

1-element_at.py
Returns the element at a given index, or None if the index is out of range.

2-replace_in_list.py
Replaces the element at a given index in a list.

3-print_reversed_list_integer.py
Prints all integers of a list in reverse order.

4-new_in_list.py
Returns a new list with the element at a given index replaced.

5-no_c.py
Returns a new string with all c and C characters removed.

6-print_matrix_integer.py
Prints a matrix of integers, row by row.

7-add_tuple.py
Returns the element-wise sum of two tuples.

8-multiple_returns.py
Returns a tuple with the length of a string and its first character.

9-max_integer.py
Returns the maximum integer in a list.

10-divisible_by_2.py
Returns a list of booleans indicating if each number is even.

11-delete_at.py
Deletes the element at a given index in a list.

12-switch.py
Swaps the values of two variables and prints the result.
