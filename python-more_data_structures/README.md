# 0x04. Python - More Data Structures: Set & Dictionary

This folder contains Python scripts that focus on working with:
- lists
- sets
- dictionaries

The tasks practice searching, replacing, unique values, set operations,
and simple dictionary manipulations.

---

## How to use

Make the files executable (optional but recommended):

```bash
chmod +x *.py
```

Run any file with:
```
./file_name.py
```
Files Description

0-square_matrix_simple.py
Returns a new matrix where each integer is squared.

1-search_replace.py
Returns a new list with all occurrences of a value replaced by another.

2-uniq_add.py
Returns the sum of all unique integers in a list.

3-common_elements.py
Returns the common elements between two sets.

4-only_diff_elements.py
Returns elements that are in one set or the other, but not both.

5-number_keys.py
Returns the number of keys in a dictionary.

6-print_sorted_dictionary.py
Prints a dictionary sorted by keys (key: value).

7-update_dictionary.py
Adds or updates a key in a dictionary with a given value.

8-simple_delete.py
Deletes a key from a dictionary if it exists and returns the dictionary.

9-multiply_by_2.py
Returns a new dictionary with all values multiplied by 2.

10-best_score.py
Returns the key with the highest value in a dictionary, or None if empty.

11-multiply_list_map.py
Returns a new list with all elements multiplied by a number using map().

12-roman_to_int.py
Converts a Roman numeral string into its integer value.
