The ``MyList`` class
====================

Basic test
----------

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Test with negative number (unsorted list)
-----------------------------------------

>>> my_list2 = MyList()
>>> my_list2.append(10)
>>> my_list2.append(-3)
>>> my_list2.append(7)
>>> my_list2.append(0)
>>> print(my_list2)
[10, -3, 7, 0]
>>> my_list2.print_sorted()
[-3, 0, 7, 10]
>>> print(my_list2)
[10, -3, 7, 0]
