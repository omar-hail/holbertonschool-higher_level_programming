# 0x01. Python - if/else, loops, functions

This project contains beginner-level Python tasks designed to practice:
- Conditional statements (`if / elif / else`)
- Loops
- Working with ASCII and `ord()`
- String formatting
- Creating simple functions
- Handling integer and string logic
- Printing formatted output

All files follow **pycodestyle** (PEP8) and are written for **Python 3**.
