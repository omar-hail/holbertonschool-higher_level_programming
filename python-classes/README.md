# Python - Classes and Objects (Square)

## Features Added Across Tasks

- Private attributes (`__size`, `__position`)
- Input validation with specific error messages
- Properties and setters for clean attribute access
- Area computation
- Visual printing of the square using `#`
- Position-based shifting of printed output
